
# Fahad Shaikh // GPU Portfolio

High-performance GPU Systems & AI Research portfolio. Built with React and a custom hardware-accelerated SVG engine.

## Features
- **GPU Die Visualization**: Interactive SVG map of NVIDIA's Hopper architecture.
- **Neural Skills Graph**: Dynamic visualization of ML/Systems expertise.
- **GitHub Integration**: Live contribution tracking.
- **Responsive Design**: Optimized for 4K displays and mobile devices.

## Tech Stack
- React.js
- Tailwind-style Vanilla CSS
- Custom Serialized Asset Bundler
- GitHub Activity API

## Deployment
Deployed via [Vercel](https://vercel.com). Simply push to the `main` branch to trigger a redeploy.
